chanyue-cms

基于express+mysql的一款轻量级高质量cms管理系统

开始使用 码云 下载

大道至简

基于 express 路由框架，拒绝复杂设计模式。

独有特色

轻量、灵活、稳定、高性能

SEO标准

专注于`seo`,伪静态`html`和拼音导航，灵活设置关键词和描述

灵活

碎片功能，支持零碎文案配置，方便各类灵活文案配置。

高扩展

支持扩展模型，字段配置，可动态生成表，超强扩展，可进行多领域网站开发，

高性能模板

由art-template 模板引擎带来的简单易上手体验，无需二次开发即可定义一切
